# Security Policy

## Supported Versions

Only the latest version of **Is It Open** is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please do not disclose it publicly. Instead, report it by opening a private security advisory on GitHub or by contacting the maintainers directly.

We will acknowledge your report and provide an estimated timeframe for a fix.
